# Predictive Analytics Using Historical Data

This project uses machine learning and historical airline passenger data to forecast future passenger trends.

## Features
- Data preprocessing
- Data visualization
- Predictive analytics
- Linear regression model
- Streamlit dashboard

## Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn
- Streamlit

## Run Project

```bash
streamlit run app.py
